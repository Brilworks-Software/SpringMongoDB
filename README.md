# Spring Boot MongoDB

Spring Boot CRUD application with MongoDB using the Java Mongo driver.

## Tech Stack

- **Framework:** Spring Boot
- **Database:** MongoDB
- **Language:** Java
- **Build:** Maven

## Features

- MongoDB CRUD operations via Java driver
- REST API endpoints
- Document-based data modeling

## Getting Started

```bash
# Ensure MongoDB is running locally
mvn clean install
mvn spring-boot:run
```
